# PURPOSE

Mockingbird converts approved audio tracks into streaming-ready formats.

Responsibilities:

```
1. Listen for track.approved (strict contract: sourceStorage, objectKey)
2. Download original audio from object storage (bucket/key)
3. Transcode into 128 and 320 kbps MP3 variants
4. Generate HLS (master playlist + variant playlists + segments)
5. Upload transcoded files to MinIO
6. Emit track.transcoding.completed and track.hls.generated
```

Mockingbird **does NOT**:

```
analyze audio
run stereo (AI approval)
persist HLS to storage
serve playback
```

# EVENTS CONSUMED

```
track.approved
```

**Strict payload contract:**

```json
{
  "trackId": "uuid",
  "objectKey": "uploads/{trackId}/original.mp3",
  "sourceStorage": {
    "bucket": "uploads",
    "key": "uploads/{trackId}/original.mp3"
  },
  "decision": "approved",
  "reason": "AI approval reason",
  "approvedAt": "2026-03-15T12:00:00Z"
}
```

**Contract validation:** TrackApprovedConsumer validates `sourceStorage.bucket`, `sourceStorage.key`, and `objectKey === sourceStorage.key`. On violation, emits `track.transcoding.failed` with `TRACK_APPROVED_CONTRACT_INVALID` and does not transcode.

# EVENTS EMITTED

```
track.transcoding.started
track.transcoding.completed
track.hls.generated
track.transcoding.failed
```

Example `track.transcoding.completed`:

```json

  "trackId": "uuid",
  "variants": [
    { "bitrate": 128, "bucket": "transcoded", "key": "transcoded/{trackId}/128.mp3" },
```

```json
{
  "trackId": "uuid",
  "masterPlaylist": "/tmp/hls/{trackId}/master.m3u8",

      "bitrate": 128,
```

# PORTS

| Port | Responsibility |
|------|----------------|
| StoragePort | download({ bucket, key }) → local path; upload(objectKey, filePath) |
| TranscoderPort | transcode(inputFile, bitrate) → output path; generateHls(inputFile, bitrate, outputRoot) → { playlist, segmentsDir } |
| MockingbirdEventBusPort | Typed event bus for MockingbirdEventMap |

# ADAPTERS

| Port | Adapter |
|------|---------|
| StoragePort | MinIO via S3 client (download from source bucket, upload to transcoded bucket) |
| TranscoderPort | FFmpegTranscoderAdapter (FFmpeg CLI) |
| MockingbirdEventBusPort | NatsEventBusAdapter |

# TRANSCODING TARGETS

- **128 kbps MP3** — `transcoded/{trackId}/128.mp3`
- **320 kbps MP3** — `transcoded/{trackId}/320.mp3`
- **HLS** — generated under `/tmp/hls/{trackId}/` (master.m3u8, variant playlists, segments)

---

# STORAGE STRATEGY

**Buckets:** `uploads` (source), `transcoded` (MP3 outputs)

**Layout:**

```
uploads/{trackId}/original.mp3          # Source (from Soundgarden / Shinod AI)
transcoded/{trackId}/128.mp3
transcoded/{trackId}/320.mp3
```

**HLS** is written to local filesystem (`/tmp/hls`) and passed to Hybrid Storage via `track.hls.generated` (local paths). Hybrid Storage persists to MinIO.

# CONTRACT VALIDATION

TrackApprovedConsumer:

1. Validates `payload.sourceStorage?.bucket` and `payload.sourceStorage?.key` exist
2. Validates `payload.objectKey` exists and equals `payload.sourceStorage.key`
3. On failure: emits `track.transcoding.failed` with `errorCode: TRACK_APPROVED_CONTRACT_INVALID`, returns early (no transcode)
