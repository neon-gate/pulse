# PURPOSE

Hybrid Storage persists HLS streaming artifacts to object storage.

Responsibilities:

```
1. Listen for track.hls.generated
2. Persist master playlist, variant playlists, and segments to MinIO
3. Emit track.hls.stored on success
4. Emit track.hls.failed on persist failure
```

Hybrid Storage **does NOT**:

```
generate HLS
transcode audio
serve playback
own transcoding
```

Those belong to Mockingbird and the transcoder.

# EVENTS CONSUMED

```
track.hls.generated
```

Example payload:

```json
{
  "trackId": "uuid",
  "masterPlaylist": "/tmp/hls/{trackId}/master.m3u8",
  "variants": [
    {
      "bitrate": 128,
      "playlist": "/tmp/hls/{trackId}/128/index.m3u8",
      "segmentsDir": "/tmp/hls/{trackId}/128"
    },
    {
      "bitrate": 320,
      "playlist": "/tmp/hls/{trackId}/320/index.m3u8",
      "segmentsDir": "/tmp/hls/{trackId}/320"
    }
  ],
  "generatedAt": "2026-03-15T12:00:00Z"
}
```

---

# EVENTS EMITTED

```
track.hls.stored
track.hls.failed
```

Example `track.hls.stored`:

```json
{
  "trackId": "uuid",
  "baseKey": "tracks/{trackId}/hls",
  "manifestKey": "tracks/{trackId}/hls/master.m3u8",
  "storedAt": "2026-03-15T12:00:00Z"
}
```

Example `track.hls.failed`:

```json
{
  "trackId": "uuid",
  "errorCode": "HLS_PERSIST_FAILED",
  "message": "Error description"
}
```

# PORTS

| Port | Responsibility |
|------|----------------|
| StoragePort | putObject({ key, body, contentType }) for S3/MinIO |
| HybridStorageEventBusPort | Typed event bus for HybridStorageEventMap |

---

# ADAPTERS

| Port | Adapter |
|------|---------|
| StoragePort | MinioStorageAdapter (@aws-sdk/client-s3) |
| HybridStorageEventBusPort | NatsEventBusAdapter (or NoopEventBusAdapter when NATS off) |
