# PURPOSE

Backstage provides real-time visibility into the ingestion pipeline.

It consumes pipeline lifecycle events and streams them to developer tooling and monitoring UIs.

Initial version (V3):

```
1. Mock pipeline events
2. Broadcast via WebSocket (Socket.IO)
3. Pulse reasoning UI subscribes and renders in real time
4. Events displayed with playful, human narrative
```

Future versions will:

```
consume NATS events
persist pipeline state
expose monitoring APIs
```

Backstage **does NOT**:

```
process audio
run AI cognition
orchestrate services
store media files
emit pipeline commands
```

Backstage is strictly a **pipeline event broadcaster**.

# EVENTS CONSUMED

Backstage subscribes to events defined across the platform.

Events follow the abstractions in `@pack/kernel` — **Backstage must not invent new event names**.

Examples:

```
track.upload.received
track.upload.validated
track.upload.stored
track.uploaded

track.petrified.generated
track.petrified.song.unknown

track.fort-minor.started
track.fort-minor.completed

track.stereo.started

track.approved
track.rejected

track.transcoding.started
track.transcoding.completed

track.ready
track.upload.failed
```

The service treats events generically so new pipeline steps can be added without rewriting core logic.

---

# EVENT PAYLOAD SOURCE

Backstage receives events using the kernel event abstraction.

Conceptual structure:

```
Event<TPayload>
```

Example kernel event payload:

```json
{
  "subject": "track.uploaded",
  "timestamp": "2026-03-15T12:00:00Z",
  "data": {
    "trackId": "uuid"
  }
}
```

Backstage maps kernel events to WebSocket messages. It does not alter the event schema.

# USE CASE

### broadcast-pipeline-event.use-case.ts

```ts
import { EventStreamPort, PipelineEventPayload } from "../ports/event-stream.port"

export class BroadcastPipelineEventUseCase {

  constructor(
    private readonly stream: EventStreamPort
  ) {}

  async execute(input: {
    trackId: string
    eventType: string
    payload: Record<string, unknown>
  }): Promise<void> {

    const payload: PipelineEventPayload = {
      type: "pipeline.event",
      event: input.eventType,
      trackId: input.trackId,
      timestamp: new Date().toISOString(),
      payload: input.payload ?? {}
    }

    await this.stream.broadcast(payload)
  }
}
```

# WEBSOCKET ADAPTER

### socketio-event-stream.adapter.ts

```ts
import type { Server } from "socket.io"
import type {
  EventStreamPort,
  PipelineEventPayload
} from "../../application/ports/event-stream.port"

export class SocketIOEventStreamAdapter implements EventStreamPort {

  constructor(private readonly gateway: { server: Server }) {}

  async broadcast(event: PipelineEventPayload): Promise<void> {
    if (this.gateway.server) {
      this.gateway.server.emit("pipeline.event", event)
    }
  }
}
```
